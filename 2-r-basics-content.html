2. R言語の基礎

基本的な演算と関数

Rは強力な計算機能を持っています。以下に基本的な演算と関数の例を示します：

# 基本的な算術演算
5 + 3   # 加算
10 - 4  # 減算
6 * 7   # 乗算
20 / 5  # 除算
2 ^ 3   # べき乗

# 基本的な関数
sqrt(16)   # 平方根
abs(-7)    # 絶対値
log(10)    # 自然対数
exp(2)     # 指数関数
round(3.7) # 四捨五入

データ型（数値、文字列、論理値）

Rには以下の主要なデータ型があります：

1. 数値 (numeric)

x <- 10.5  # 小数
y <- 5     # 整数
class(x)   # "numeric"
class(y)   # "numeric"

2. 文字列 (character)

name <- "John Doe"
class(name)  # "character"

3. 論理値 (logical)

is_true <- TRUE
is_false <- FALSE
class(is_true)  # "logical"

ベクトルと行列の操作

ベクトル

ベクトルは同じデータ型の要素の集まりです。

# ベクトルの作成
numbers <- c(1, 2, 3, 4, 5)
fruits <- c("apple", "banana", "cherry")

# ベクトルの操作
numbers[3]          # 3番目の要素にアクセス
numbers * 2         # すべての要素を2倍
length(fruits)      # ベクトルの長さ
sum(numbers)        # 合計
mean(numbers)       # 平均

行列

行列は2次元のデータ構造です。

# 行列の作成
m <- matrix(1:9, nrow = 3, ncol = 3)

# 行列の操作
m[2, 3]             # 2行3列の要素にアクセス
t(m)                # 転置
m * 2               # すべての要素を2倍
colSums(m)          # 列ごとの合計
rowMeans(m)         # 行ごとの平均

データフレームの基本

データフレームは、異なるデータ型の列を持つことができる2次元のデータ構造です。生物統計学でよく使用されます。

# データフレームの作成
df <- data.frame(
  name = c("Alice", "Bob", "Charlie"),
  age = c(25, 30, 35),
  height = c(165, 180, 175)
)

# データフレームの操作
df$name             # name列にアクセス
df[2, "age"]        # 2行目のage列にアクセス
nrow(df)            # 行数
ncol(df)            # 列数
summary(df)         # 要約統計量

注意: これらは基本的な操作の一部にすぎません。Rには多くの高度な機能と関数があり、データ分析や統計処理を効率的に行うことができます。
