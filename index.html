1. はじめに

Rとは何か

Rは、統計計算とグラフィックスのためのプログラミング言語および環境です。生物統計学、データ分析、機械学習など、幅広い分野で使用されています。以下がRの主な特徴です：

  • オープンソースで無料
  • 統計解析に特化した豊富な機能
  • 拡張性が高く、多数のパッケージが利用可能
  • データ可視化のための強力なツール
  • アクティブなコミュニティサポート

Rのインストール方法

  1. R公式ウェブサイトにアクセス
  2. 「download R」をクリック
  3. お使いのオペレーティングシステム（Windows, Mac, Linux）を選択
  4. 最新版のRをダウンロードしてインストール

注意: インストール時にデフォルト設定を変更する必要はありません。

RStudioの紹介と基本的な使い方

RStudioは、R言語のための統合開発環境（IDE）です。コードの書きやすさ、データの可視化、プロジェクト管理などの機能を提供し、R言語の使用をより効率的にします。

RStudioのインストール

  1. RStudio公式ウェブサイトにアクセス
  2. RStudio Desktopの無料版をダウンロード
  3. ダウンロードしたインストーラーを実行

RStudioの基本的な使い方

  1. RStudioを起動すると、4つのペインに分かれた画面が表示されます：
    • 左上: ソースエディタ（Rスクリプトを書く場所）
    • 左下: コンソール（コマンドを直接実行する場所）
    • 右上: 環境／履歴（作成した変数や実行履歴を確認）
    • 右下: ファイル／プロット／パッケージ／ヘルプ（多目的ペイン）
  2. 新しいRスクリプトを作成: File > New File > R Script
  3. コードの実行:
    • 1行実行: カーソルを置いて Ctrl+Enter（Mac: Cmd+Enter）
    • 複数行実行: 範囲選択して Ctrl+Enter（Mac: Cmd+Enter）
  4. パッケージのインストールと読み込み: 
    # パッケージのインストール
install.packages("ggplot2")

# パッケージの読み込み
library(ggplot2)
  5. ヘルプの利用:
    • 関数のヘルプを見る: ?関数名（例: ?mean）
    • パッケージのヘルプを見る: ヘルプペインの検索ボックスを使用
